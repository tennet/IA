📝 Prompt Engineering
L’arte di fare domande giuste all’IA

Scrivere buoni prompt è fondamentale per ottenere risposte utili e precise. Questo processo è noto come prompt engineering.

Casi d’uso:
• Prompt generico: “Scrivi un testo sull’energia” → Risposta generica
• Prompt migliorato: “Scrivi 3 vantaggi dell’energia solare, in stile informativo per un volantino.” → Risposta mirata
Curiosità: Alcuni professionisti lavorano solo come prompt engineer!
Pro Tip: Sii specifico, dai istruzioni sul formato, tono e pubblico.
Esercizio:
Scrivi un prompt su un argomento che conosci. Miglioralo in 3 versioni rendendolo sempre più chiaro e mirato.
